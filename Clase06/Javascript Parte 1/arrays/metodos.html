splice()

El método splice () se puede utilizar para agregar nuevos elementos. El primer parámetro (2) define la posición en la que se deben agregar nuevos elementos . El segundo parámetro (1) define cuántos elementos deben eliminarse . El resto de parámetros ("Limón", "Kiwi") definen los nuevos elementos a añadir . el metodo splice devuelve los elementos eliminados.
